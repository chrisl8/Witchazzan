Mobile Controls

Tap on the left side of the screen to activate the blue stick. Use this to move your character.
Tap on the right side of the screen to activate the red stick. Use this to change settings and perform actions.
For the Red stick:
  • Tap briefly to fire spell.
  • Drag in a direction to change spells or perform actions.
  • The action will display in the upper left.
  • Release to perform action, or return to center to cancel.

In a mobile browser the sticks can be activated anywhere in the bottom half of the browser window. If you add the game to your Home Screen as an "App" then they can be activated anywhere on the screen.
The sticks will follow your finger, so you can drag and use them anywhere on the screen once you activate them.

Instructions

Premise

  • This is an online Multi-player game. When you click "Start Game" the game will load and connect to the server. Without the server, the game will not function.
  • For reasons, the server is called the Small Hadron Cooperator.

Remote Shadow

  • You will quickly notice a darkened instance of your sprite that follows you. This is not another player, it is you as seen by the server and other players. It provides a visual indication of the lag between you and the remote world.

Directional Controls

  • wasd or Arrow Keys
    • Hold down Shift to "Sprint" which bypasses acceleration. Max speed is still the same.

Spells

  • The number keys, 1-2 at the moment, pick which spell is active.
  • You pick which spell to assign to which key in the "Setup" section above.
  • The space bar fires the currently active spell.

Other Controls

  • Space Bar - Fire
  • Chat/Command Input
    • There are three ways to open the Chat input box, which also doubles as the Command input box:
      • t
      • Enter
      • / - starts you out with a slash already in the input for sending commands.
  • Chat Log
    • The Chat Log opens and closes automatically when you open or close the Chat/Command Input box.
    • The Chat Log also opens when you receive a message from someone or something.
    • You can also Close the Chat Log with the Escape key, or by pressing 'l' if it opened by itself.
    • l - Will also open (and close) the Chat Log at any time without opening the Chat Input box.
  • i - teleport out to your Inventory
    • When you do this you leave the scene you are in. Upon exiting your inventory you return to the same location in the same scene that you left from.
  • h - return to the default initial scene NOTE: Only Admins have access to this command. Everybody else has to walk!
  • o - turn dot Trails on/off for debugging
  • p - "Pause" returns you to this screen
    • This will also reload and pause your game, but remember the game is online multi-player, so it goes on without you.

Chat/Command Input

  • Any text entered is broadcast as chat text to all other players.
  • Esc - Close Chat/Command input box
  • / - send commands instead of chat
  • Send /help to get a list of available commands.
