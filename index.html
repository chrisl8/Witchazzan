Instructions

Directional Controls

  • wasd or Arrow Keys
    • Hold down Shift to "Sprint" which bypasses acceleration. Max speed is still the same.

Keys

  • Space Bar - Fire
  • c - open the Chat/Command input box
  • l - open/close the chat Log when chat input is not open
  • o - return to Opening scene
  • t - turn dot Trails on/off for debugging
  • h - return to this Help screen
    • This will also reload and pause your game.

Chat/Command Input

  • Any text entered is broadcast as chat text to other players.
  • Esc - Close Chat/Command input box
  • / - send commands instead of chat
  • /who - list who is online
  • /run <command> - run built in functions

Touch Controls

  • Use Finger or Mouse pointer like a finger to move
    • Distance from center of starting point determines speed
  • Tap with TWO fingers to fire
  • Tap with THREE fingers to open/close the chat/command input
